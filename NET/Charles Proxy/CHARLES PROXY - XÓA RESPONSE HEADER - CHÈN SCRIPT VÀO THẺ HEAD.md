# CHARLES PROXY - XÓA RESPONSE HEADER - CHÈN SCRIPT VÀO THẺ HEAD

### BƯỚC 1: CÀI ĐẶT PHẦN MỀM CHARLES PROXY

### BƯỚC 2: INSTALL CHARLES ROOT CERTIFICATE

```code
HELP -> SSL PROXYING -> INSTALL CHARLES ROOT CERTIFICATE -> TIẾN HÀNH CÀI ĐẶT CERTIFICATE VÀO HỆ THỐNG
```

### BƯỚC 3: SAVE CHARLES ROOT CERTIFICATE

```code
HELP -> SSL PROXYING -> SAVE CHARLES ROOT CERTIFICATE -> LƯU CERTIFICATE THEO ĐỊNH DẠNG BINARY CERTIFICATE (.CER)
```

### BƯỚC 4: KÍCH HOẠT WINDOWS PROXY

```code
PROXY -> KÍCH HOẠT -> WINDOWS PROXY (CTRL + SHIFT + P)
```

### BƯỚC 5: START SSL PROXYING

```code
PROXY -> START SSL PROXYING (CTRL + L)
```

### BƯỚC 6: SSL PROXYING SETTINGS

```code
PROXY -> SSL PROXYING SETTINGS (CTRL + SHIFT + L)
- CHỌN THẺ SSL PROXYING -> ENABLE SSL PROXYING
- THÊM LOCATION CẦN BẮT GÓI TIN VÀO DANH SÁCH INCLUDE. Example -> "ABC.COM" VÀ "*.ABC.COM"
```

### BƯỚC 7: NẾU MÁY TÍNH CÓ CÀI ANTIVIRUS THÌ CẦN THÊM CERTIFICATE (.CER) CỦA CHARLES PROXY VÀO DANH SÁCH TIN TƯỞNG CỦA PHẦN MỀM ANTIVIRUS

### BƯỚC 8: REWRITE SETTINGS

```code
TOOLS -> REWRITE (CTRL + ALT + R) -> ENABLE REWRITE
- THÊM PROFILE -> ABC.COM
- THÊM LOCATION -> "HTTP://ABC.COM" VÀ "HTTP://*.ABC.COM"
- THÊM LOCATION -> "HTTPS://ABC.COM" VÀ "HTTPS://*.ABC.COM"
- THÊM RULE -> TYPE = "REMOVE HEADER" -> WHERE = "RESPONSE" -> MATCH -> NAME = "CONTENT-SECURITY-POLICY"
- THÊM RULE -> TYPE = "REMOVE HEADER" -> WHERE = "RESPONSE" -> MATCH -> NAME = "X-CONTENT-SECURITY-POLICY"
- THÊM RULE -> TYPE = "BODY" -> WHERE = "RESPONSE" -> MATCH -> VALUE = "</HEAD>" -> REPLACE -> VALUE = "<SCRIPT SRC='ABC.JS'></SCRIPT></HEAD>"
```

### BƯỚC 9: MAP LOCAL SETTINGS

```code
TOOLS -> MAP LOCAL (CTRL + ALT + L) -> ENABLE MAP LOCAL
- NHẤN NÚT ADD ĐỂ THÊM LOCATION VÀ LOCAL PATH VÀO DANH SÁCH
- MAP FROM -> HOST = ABC.COM -> PATH = ABC.JS
- MAP TO -> LOCAL PATH = CHỌN FILE ABC.JS ĐANG LƯU TRỮ TRÊN HỆ THỐNG
```
